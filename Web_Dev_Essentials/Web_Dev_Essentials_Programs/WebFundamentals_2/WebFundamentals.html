Web 3.0 differences from Web 2.0 and 1.0

Web 1.0 was the beginning of the world wide web where information was read-only, web 2.0 allows users to read and write their own information
an example is social media. Web 3.0 introduces a new way of interacting with the world wide web in such a way that the Internet itself assists
user when they are search for information which is machine learning.Users will have more security when storing data and sharing information.

Functional Differences of Front and Back End

Front-end has got to do with how the users interact with the website and
Back-end is what the user doesn't see which is how the website is requested from a server and stores users data.

Process of a URL search

Each url is a unique address that is stored on a server that can be visited when entering it in a search engine which fetches
the requested url recieves a response from the server and takes you the respective website by translating the website
so that it can be displayed on the users device.

What to consider before using a image on your website

Before putting an image on your website you should take the following into consideration

  • Is the image appropriate to the topic of the site
  • Will the images size appropriate for the site
  • Do you have the legal right of permission to display the image of your site
  • Is the image from a local source or is it from the internet
  • You have actual proof that the image is in a public domain
